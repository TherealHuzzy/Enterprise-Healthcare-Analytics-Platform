# Patient Administration System (PAS)

## Purpose

The Patient Administration System manages patient demographics, appointments, referrals, consultants and clinics. It serves as the primary operational system for outpatient administration.

## Core Entities

- Patient
- Appointment
- Consultant
- Clinic
- Referral
- Department
- Appointment Status
- Location

## Relationships

- One Patient can have many Appointments.
- One Consultant can have many Appointments.
- One Clinic can have many Appointments.
- One Referral can generate many Appointments.
